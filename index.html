Installation 4chan Reddit

The styles work in Firefox 19.x, as far as I know.

Install the styles

Firefox:
  1. If you don't have Stylish...
    1. Install Stlyish.
    2. Restart Firefox.
  2. Create a new style.
  3. Copy the code from the style of your choice into the new style.
  4. Save and apply.
